# URL Shortener with QR & Click Tracking

A simple and efficient URL Shortener built using **FastAPI**, with support for:
  
  Custom Aliases  
  Expiry Date for Short URLs  
  QR Code Generation  
  Click Analytics / Stats  
  Error Handling & Logs


##  Features

| Feature | Description |
|--------|-------------|
| Short URL | Convert long URLs into short and shareable links |
| Custom Alias | Choose your custom keyword instead of random code |
| Expiry | Automatically disable links after selected days |
| QR Code | Easily open links on mobile devices |
| Click Tracking | Counts how many times each short link is visited |
| Statistics Page | View analytics of each short link |


##  Tech Stack

| Layer | Technology |
|------|------------|
| Backend | FastAPI |
| Database | SQLite + SQLAlchemy ORM |
| Frontend | HTML, CSS, JavaScript |
| QR Generation | Python `qrcode` library |
| Testing | Pytest |
